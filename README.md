# enterprise-wide area network

A wide area network spanned across New York, Michigan, Ohio, and New Jersey:

- ASA Firewall 5505 + 5510
- Cisco Catalyst 3560
- Cisco Catalyst 2960-X
- Meraki MR66 Access Points
- Meraki Cloud Controller
- Windows Server 2012 Active Directory
- Windows Server 2012 Radius Server
- Verizon, Optimum, and AT&T 1G Gigabit fiber lines
  

## Features

- Consolidate access to internal resources and services
- Seamless Wireless Logins across remote sites
- Improved Security and network Connectivity
- Seamless Guest User Access for customers and external stakeholders
- Faster internet
